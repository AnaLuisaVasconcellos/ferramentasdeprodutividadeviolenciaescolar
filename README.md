# ferramentasdeprodutividadeviolenciaescolar
Ferramentas de Produtividade para analisar retratos da violência escolar nas redes sociais. As discussões online sobre tiroteios escolares geram grande volume de comentários carregados de emoção, julgamento e interpretações políticas. Proponho um modelo de automação para analisar comentários através de linguagens de programação.
